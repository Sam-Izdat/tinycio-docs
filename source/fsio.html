tinycio.fsio.load_image(fp, graphics_format=GraphicsFormat.UNKNOWN)[source]

Load image file as [C, H, W] float32 PyTorch image tensor. If the graphics format is not specified, we take a guess. Unsigned int data is automatically converted to float32 in [0, 1] range. Floating point data is converted to float32 if needed, but is otherwise undisturbed.

Parameters:

  • fp (str) – File path to load from (including name and extension).

  • graphics_format (GraphicsFormat) – Graphics format of the image.

Returns:

[C, H, W] sized image tensor

Return type:

torch.Tensor

tinycio.fsio.save_image(im, fp, graphics_format=GraphicsFormat.UNKNOWN)[source]

Save [C, H, W] float32 PyTorch image tensor as image file. File format is inferred from extension by imageio. If a graphics format is not specified, we take a guess.

Warning

This will overwrite existing files.

Parameters:

  • im (torch.Tensor) – Image tensor.

  • fp (str) – File path for saving the image (including name and extension).

  • graphics_format (GraphicsFormat) – Graphics format of the image.

Returns:

True if successful

Return type:

bool
